## Pré-requisitos:

- Node.js
- Yarn

## Como utilizar:
### App

Clone o repositório
```bash
# Abra o terminal e digite esse comando ou faça download do projeto
$ git clone git@github.com:Fekleite/letmeask.git
```

Entre na pasta
```bash
$ cd letmeask
```

Instale as dependências
```bash
$ yarn install
```

Inicie a aplicação
```bash
$ yarn start
```
